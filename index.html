Мобильный веб не дотягивает по UX

  • Среднее время загрузки ~ 17 секунд
  • Bounce rate выше
  • Количество посещенных страниц меньше

Критический путь отрисовки

Почему у AMP все быстро?

  • Асинхронный JavaScript
  • Рсположение элементов предопределено
  • Сторонний JavaScript запускается в iframe
  • CSS выдаем инлайном
  • Ничего не останавливает браузер от рендеринга (ну если только вы подключаете шрифты)
  • Lazy-load ресурсов (Viewport)
  • Prerendering при поиске в Google
  • Кэширование
